QR Code Generation using Python.
QR codes are a popular way to encode data in a visual format that can be scanned using smartphones or other devices. In Python, the qrcode library makes generating QR codes both simple and highly customizable.This script takes a link of any URL and generate a QR code corresponding to it.
Prerequisites:
     Python 3 and above versions
Libraries Used are qrcode and IPython
To install required external modules
 Run pip install qrcode
How to run the script
* Provide your desired URL in the script 
* Execute python3 QR_code_generator.py
